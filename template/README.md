# vue-ssr-template

## 脚手架 支持 vue 同时输出ssr 跟spa（接口请求一定要用vuex）
##  路由使用 history 模式

## 命令：

   dev       : ssr 开发模式

   build:ssr : ssr编译模式

   start: ssr 发布测试

   client:dev : spa 开发模式

   build:spa : spa编译模式

   test:client: spa 发布测试

   mock : 模拟json 数据

   analyz:  文件分析
